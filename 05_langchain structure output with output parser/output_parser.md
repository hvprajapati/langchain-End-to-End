# Output Parsers in LangChain

## What are Output Parsers?

Output Parsers are components in LangChain that **convert the raw text generated by an LLM into a structured format** that your application can easily use.

Instead of manually parsing the LLM's response, Output Parsers do it for you.

---

## Why Do We Need Output Parsers?

### Without Output Parser

```
User
   │
   ▼
LLM
   │
   ▼
Raw Text
   │
   ▼
Developer manually parses the text ❌
```

Problems:

- Inconsistent output
- Difficult to parse
- Error-prone
- Extra coding required

---

### With Output Parser

```
User
   │
   ▼
LLM
   │
   ▼
Output Parser
   │
   ▼
Structured Output ✅
```

Benefits:

- Consistent output
- Easy to process
- Production-ready
- Less parsing code

---

# Types of Output Parsers

## 1. StrOutputParser

Returns plain text.

```
LLM
 │
 ▼
"LangChain is a framework..."
```

Use When:

- Chatbots
- Summarization
- Translation
- General Q&A

---

## 2. JsonOutputParser

Returns JSON.

```
LLM
 │
 ▼
{
    "name": "Hardik",
    "age": 22
}
```

Use When:

- APIs
- Frontend applications
- Database insertion
- JSON responses

---

## 3. StructuredOutputParser

Returns structured data based on predefined fields.

```
LLM
 │
 ▼
Structured Object
```

Use When:

- Forms
- Data extraction
- Resume parsing
- Information extraction

---

## 4. PydanticOutputParser

Returns a validated Pydantic object.

```
LLM
 │
 ▼
Person(
    name="Hardik",
    age=22
)
```

Use When:

- Production applications
- FastAPI
- AI Agents
- Data validation
- Complex schemas

---

# Which Parser Should You Use?

| Parser | Output | Best For |
|----------|---------|----------|
| StrOutputParser | String | Chatbots, Text Generation |
| JsonOutputParser | JSON | APIs, Frontend, Databases |
| StructuredOutputParser | Structured Fields | Data Extraction |
| PydanticOutputParser | Pydantic Object | Production Applications |

---

# Learning Order

```
StrOutputParser
        ↓
JsonOutputParser
        ↓
StructuredOutputParser
        ↓
PydanticOutputParser
```

# One-Line Summary

> **Output Parsers transform raw LLM responses into structured, predictable formats so they can be easily used in real-world applications.**
